# SentinelIQ | Real-Time AI Incident Monitoring 🛰️
SentinelIQ is an Applied AI platform engineered to aggregate and analyze emergency incidents in the Waterberg region.

### 🚀 Features:
* **RAG Architecture**: Uses Retrieval-Augmented Generation to ground AI responses in real-time news.
* **Live Ingestion**: Connects to NewsAPI for automated incident discovery.
* **Geospatial Dashboard**: Interactive mapping with Leaflet.js.

### 🛠️ Stack:
Python (FastAPI), JavaScript, Render (Backend), Hugging Face (Frontend).
